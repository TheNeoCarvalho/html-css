# GIT

## Iniciando o Projeto

```
git init
```

## Criando um Branch

```
git checkout -b nome
```

## Verificar estatus dos arquivos

```
git status
```

## Adicionando arquivos

```
git add arquivo
```

ou

```
git add .
```

## Comitando

```
git commit -m "msg"
```

## Trocando de Branch

```
git checkout nome
```

## Criando uma Branch

```
git checkout -b nome
```
